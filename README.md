# 数组二叉树
数组二叉树，EE2331.

文件导入测试集二叉树，使用纯数组的方法进行二叉树统计层高，叶数量，和单元。


思路：


2个指针（其实用数字代替了），一个名字叫index指向当前单元，一个名叫point指向她的孩子，


每次index更新（+1），point都会+2，其中会进行point是否越界等操作。


最后，剩下没用分配到孩子的，他们就是孩子，所以进行一个遍历，看看还有多少个孩子。


-1表示空。
